# Claude Code Flow

Autonomous workflow for Claude Code.

This repository is the public entry point.

It works like a second pilot for development: it helps plan, guide, and carry tasks step by step.

## What this is

This project is a personal development system for managing coding tasks, keeping the workflow organized, and moving projects forward with less friction.

## Why it exists

It is built for myself first: to launch my own system, keep structure, and create a reusable base for future work.

## How to use

Use this repository as the main public starting point for the workflow.

## Core idea

Think of it as a second pilot for development:
- plan the task;
- guide the process;
- move step by step;
- keep the work structured.

## Status

Early stage / personal foundation.
